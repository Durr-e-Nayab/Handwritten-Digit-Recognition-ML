# Handwritten Digit Recognition ✍️

This project is part of my internship at **SoftGrowTech**. It uses Machine Learning to classify handwritten digits from the MNIST dataset.

## 🚀 Features
- **Model**: Support Vector Machine (SVM)
- **Library**: Scikit-Learn, Matplotlib, Seaborn
- **Accuracy**: 97% (High Performance)

## 📊 Results
- **Images Predicted**: The model correctly identifies digits with high precision.
- **Performance Report**: Achieved a 0.97 F1-score across most classes.

## 🛠️ How to run
1. Clone the repo.
2. Install requirements: `pip install scikit-learn matplotlib seaborn`
3. Run: `python main.py`
